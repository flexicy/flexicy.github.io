删除搜索二叉树

给定一个二叉搜索树的根节点 root 和一个值 key，删除二叉搜索树中的 key 对应的节点，并保证二叉搜索树的性质不变。返回二叉搜索树（有可能被更新）的根节点的引用。
一般来说，删除节点可分为两个步骤：
首先找到需要删除的节点； 如果找到了，删除它。 说明： 要求算法时间复杂度为 $O(h)$，h 为树的高度。
删除搜索二叉树，我理解的有三种方案 第一种：该删除的没有子树直接删除 第二种：有左子和右子树，直接接上去 第三种：都有，直接把左子树接到右子树的最左子树上去，把原始右子树接上去
因为是删除同时接上去 所以递归的时候下面应该接住新节点

插入二叉搜索树

注意，可能存在多种有效的插入方式，只要树在插入后仍保持为二叉搜索树即可。 你可以返回任意有效的结果。

注意重构可能很复杂，所以直接遍历后插入

二叉搜索树的最近公共祖先

给定一个二叉搜索树, 找到该树中两个指定节点的最近公共祖先。
百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”
说明:
所有节点的值都是唯一的。
p、q 为不同节点且均存在于给定的二叉搜索树中。

解决方法

我理解的对于这种二叉搜索树 应该是这个树的节点在遍历的时候在这两个节点中间就可以当作公共祖先，但是应该注意一个问题，这个公共祖先的节点必须是从上到下第一个遍历到的 这样才符合定义
让我想想怎么做才能从上到下的第一个符合要求：在递归的下面同时符合定义的使用 return看来可以让第一个 前面加上 if判断其他两种情况 当是其他两种情况而且有值的时候直接return那两种情况的值表示找到了不需要处理其他了 这样在代码运行的情况下发现一种符合情况的节点就会直接传输回去 同时代码是从上往下便利的 有了符合条件的再去找

二叉树的最近公共祖先

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。
百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

解决方法

对于普通二叉树的公共祖先 应该全树搜 不能直接返回 当满足左右都有的条件时候返回当前节点 否则就仅仅返回左右提供的左右节点 这样递归下去 总会到左右都有节点的时候
那么就应该在收集左右节点之后判断情况 返回节点 在递归之前的判断条件应该判断是否符合pq 同时在节点本身是公共节点的时候也不用去判断它底下的p和q了

给出一个完全二叉树，求出该树的节点个数。
示例 1：
输入：root = [1,2,3,4,5,6]
输出：6
示例 2：
输入：root = []
输出：0
示例 3：
输入：root = [1]
输出：1

解法 这种求深度个数的用后序比较好

这么看这道题的解法：完全二叉树我记得是所有的节点都有左右节点 同时没有节点层只能是最后一层 还得左边有节点 （满二叉树解法仅仅针对优化，并不是要全部替代二叉树 也替代不了）
所以两种情况 一种是底层都铺满了-对应满二叉树 这个时候是 1 2 4 2的n次方减一
第二种不是满二叉树了 所以应该递归到是满二叉树的位置上去 计算这些满二叉树的个数 之后再用普通二叉树方法求其他的加在一起

满二叉树求法：完全递归向左层数 等于完全递归向右层数 这个时候就是覆盖了满二叉树

最大二叉树

给定一个不含重复元素的整数数组。一个以此数组构建的最大二叉树定义如下：
二叉树的根是数组中的最大元素。
左子树是通过数组中最大值左边部分构造出的最大二叉树。
右子树是通过数组中最大值右边部分构造出的最大二叉树

解法

这个题我第一眼看到就是切割构造二叉树
先寻找最大的元素 之后切割左右 之后再在其中创造左右子树 应该是前序遍历比较好
找最大值这个得做个循环才行 

class Solution {//遇到的问题就是最大值查找以及建造新的节点时候的顺序
public:
    TreeNode* compare(vector<int>& nums, int a, int b) {
        if (a > b) return NULL;

        // 找到 [a, b] 范围内的最大值索引
        int index = a;
        for (int i = a; i <= b; i++) {
            if (nums[i] > nums[index]) {
                index = i;
            }
        }

        // 创建根节点
        TreeNode* root = new TreeNode(nums[index]);

        // 递归构建左子树和右子树
        root->left = compare(nums, a, index - 1);
        root->right = compare(nums, index + 1, b);

        return root;
    }

    TreeNode* constructMaximumBinaryTree(vector<int>& nums) {
        return compare(nums, 0, nums.size() - 1);
    }
};
