组合1

给定两个整数 n 和 k，返回 1 … n 中所有可能的 k 个数的组合。
示例: 输入: n = 4, k = 2 输出: [ [2,4], [3,4], [2,3], [1,2], [1,3], [1,4], ]

明白计数的开始
所说的横向遍历针对的是节点内部的横向遍历 同时在遍历的同时输出给子节点

优化 注意 return

这个优化很缺德，我理解的是因为剩下的循环里面也不够你选了，所以只选到正好能够所以的节点按顺序选择一个的界限
n- i 表示剩下的这些大于等于k - path.szie()个才行
因为 n - i >=k - path.size() 所以 i <= n- k + path.size(),
注意 因为这里表示的i等于这个值的时候 是在这个范围内有正好可以遍历的个数 所以此时i也需要在这个范围占住 所以 应该+1 因为这个i是在循环下面进行操作的

电话号码

给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。
给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母
abcdefghigklmnopqrstuvwkyz

这道题主要问题就是关于index的设置的 每加入一次 idex加1 所以从0开始相等的时候退回就行

又一个组合总和

给定一个无重复元素的数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。 candidates 中的数字可以无限制重复被选取。
所有数字（包括 target）都是正整数。
解集不能包含重复的组合。
示例 1：
输入：candidates = [2,3,6,7], target = 7,
所求解集为： [ [7], [2,2,3] ]
示例 2：
输入：candidates = [2,3,5], target = 8,
所求解集为： [ [2,2,2,2], [2,3,3], [3,5] ]

这种给定一个限制范围 之后多次在这一个集合范围里面组合的问题需要注意for循环的起始点
这次的终止条件就是求和大于等于要求的和了 如果没有达到可以无限循环下去
同时因为重复选取所以递归限制解除

组合总和终：这个给定集合有重复元素 但是提供的结果还不能有重复组合

给定一个数组 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。candidates 中的每个数字在每个组合中只能使用一次。
说明： 所有数字（包括目标数）都是正整数。解集不能包含重复的组合。

明白这种退回的思想 让path 在添加同时也能回溯 用一个固定的集合去到达目标

这道题不同点在于不可重复 同时给的区间里面还有重复 要在搜索的过程中就去掉重复组合

对于这种类型 因为给定的集合本身可以重复所以在纵向树枝上可以正常递归的时候重复 同时在同一层进行for循环的时候为了避免之前在纵向递归的时候与其重复，所以在横向遍历的时候去重操作

需要首先在横向树层进行排序 去重本身含义可以理解已经使用过的元素不能再用了

注意最后需要排序 这样才能有效把相同的树聚在一起

二叉树累加计算

从后往前写 第一个：
给出二叉搜树的根节点，该树的节点值各不相同，请你将其转换为累加树（Greater Sum Tree），使每个节点 node 的新值等于原树中大于或等于 node.val 的值之和。

因为累加树的原因所以要求从最大的开始累加，定义就是从右到左 所以遍历顺序
右边 中 左边 加入一个数值用来记录前面所有的累加值

将有序数组转换为二叉搜索树

将一个按照升序排列的有序数组，转换为一棵高度平衡二叉搜索树。
本题中，一个高度平衡二叉树是指一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1。

因为是有序树，同时是高度平衡的，所以不用担心切割的时候找不到中间位置，通过中间位置的计算切割，之后递归的时候接住就可以了

修剪二叉搜索树

给定一个二叉搜索树，同时给定最小边界L 和最大边界 R。通过修剪二叉搜索树，使得所有节点的值在[L, R]中 (R>=L) 。你可能需要改变树的根节点，所以结果应当返回修剪好的二叉搜索树的新的根节点。

每一次修建二叉树的时候有可能会发生二叉树的节点在这个边界里面，所以在删除节点的时候要求判定不违规的子节点续上。因为要续上所以返回的是有用的节点 retrun， 主要还是要判断之后跳过这个节点
