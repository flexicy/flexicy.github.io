一般来说，删除节点可分为两个步骤：

最大二叉树

给定一个不含重复元素的整数数组。一个以此数组构建的最大二叉树定义如下：
二叉树的根是数组中的最大元素。
左子树是通过数组中最大值左边部分构造出的最大二叉树。
右子树是通过数组中最大值右边部分构造出的最大二叉树

解法

这个题我第一眼看到就是切割构造二叉树
先寻找最大的元素 之后切割左右 之后再在其中创造左右子树 应该是前序遍历比较好
找最大值这个得做个循环才行 

class Solution {//遇到的问题就是最大值查找以及建造新的节点时候的顺序
public:
    TreeNode* compare(vector<int>& nums, int a, int b) {
        if (a > b) return NULL;

        // 找到 [a, b] 范围内的最大值索引
        int index = a;
        for (int i = a; i <= b; i++) {
            if (nums[i] > nums[index]) {
                index = i;
            }
        }

        // 创建根节点
        TreeNode* root = new TreeNode(nums[index]);

        // 递归构建左子树和右子树
        root->left = compare(nums, a, index - 1);
        root->right = compare(nums, index + 1, b);

        return root;
    }

    TreeNode* constructMaximumBinaryTree(vector<int>& nums) {
        return compare(nums, 0, nums.size() - 1);
    }
};
